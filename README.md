# django-telegram-otus-open-lesson
Code for otus open lesson "Django+Telegram bot"

## Step 1. Djagno + REST + PTB

Бот и Django живут отдельно и общаются через REST API

## Usage

```commandline
pip install -r requirements.txt
```

### Django 

```commandline
make migrate
```

```commandline
make server
```

### Bot

```commandline
cd bot
```

```commandline
make run
```
